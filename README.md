Notes App

An offline-first notes management Android application built using Room Database and MVVM architecture.

 Features
- Add, Edit, Delete notes (CRUD)
- Dynamic sorting (Newest / Oldest / Title)
- Search functionality
- Confirmation dialog for delete
- Timestamp-based note management
- Lifecycle-aware UI updates

 Tech Stack
- Kotlin
- MVVM Architecture
- Room Database
- LiveData
- Coroutines
- RecyclerView

Architecture
- UI Layer (Activities)
- ViewModel
- Repository
- Room DAO & Database

 Purpose
Built to strengthen understanding of local data persistence and dynamic UI updates using modern Android components.
